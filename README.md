# 课表冲突检测器

> 粘贴课表文本，AI 自动解析为结构化课表，可视化展示，一键检测时间冲突。基于 DeepSeek API，纯前端实现。

## ✨ 功能

- 🤖 **AI 智能解析** — 粘贴任意格式课表文本，DeepSeek 自动提取课程名、时间、地点
- 📅 **周课表视图** — 7 天 × 13 时段网格，课程跨时段自动合并
- ⚠️ **冲突检测** — 自动检测同一天内时间重叠的课程，红色高亮标记
- 🎨 **彩色区分** — 不同课程自动分配不同颜色，视觉一目了然
- 📱 **响应式设计** — 手机/平板/桌面端自适应

## 🚀 快速开始

1. 克隆仓库后在本地打开 `index.html`
2. 填入自己的 [DeepSeek API Key](https://platform.deepseek.com)
3. 粘贴课表文本，点「AI 解析课表」

## 📋 支持的课表格式

不限制格式，AI 会自动理解。比如：

```
高等数学 周一 8:00-9:40 A教101
大学英语 周三 10:00-11:40 B教203
线性代数 周二、周四 14:00-15:40 C教305
程序设计 周一 9:00-10:30 D教401
```

## 🏗 技术栈

| 层面 | 技术 |
|------|------|
| 前端框架 | 原生 HTML/CSS/JS（零依赖） |

| 布局 | CSS Grid |
| 部署 | 纯静态，任意 Web 服务器 / 本地打开 |
